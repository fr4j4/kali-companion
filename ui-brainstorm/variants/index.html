Kali · variantes asistente-centro

3 formas de interactuar con Kali

Punto de partida: assistant-in-center.html (orbe + proyección de texto). Las 3 variantes completan con: voz + texto, streaming, artefactos ricos (código, imágenes, fichas, enlaces, controles, markdown), historial, y estados del orbe (idle/escuchando/pensando/hablando).

orbe al centro
1 · Astra

Timeline vertical

Orbe al centro-top con texto proyectado debajo. La conversación fluye como turnos numerados hacia abajo, con artefactos inline (ficha, código, imagen, controles, enlace). Input dock abajo con voz + texto. Híbrido: escenario + conversación.
2 · Halo

Órbita radial

Orbe al centro absoluto. Los artefactos orbitan alrededor del asistente en un ring que rota lentamente. Cada nodo es una tarjeta flotante (ficha, código, imagen, control, enlace). Texto proyectado en glass pill debajo del orbe. Cero lista de chat: todo es espacio.
3 · Verse

Stage en 3 capas

Layout de 3 columnas: rail izquierdo = memoria de turnos + estado del sistema; centro = orbe + texto proyectado (la voz de Kali); rail derecho = artefactos generados colapsables. Grid blueprint de fondo, serif Instrument Serif, input tipo terminal ▸. Densidad informativa máxima.
