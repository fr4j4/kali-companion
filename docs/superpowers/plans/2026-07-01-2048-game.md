# 2048 Game Implementation Plan

> **Goal:** Implement a working 2048 game as a Kali artifact window — grid-based, turn-based, keyboard input, configurable board size, neon visual theme, score tracking.
>
> **Status:** Implemented.
> **Last updated:** 2026-07-01.

## Architecture

Single-player 2048 running entirely in the frontend as a grid-based turn-based game. The `TwentyFortyEightGame` class (`kali-web/src/games/twenty-forty-eight/twenty-forty-eight-game.ts`) manages board state, tile merging, spawn logic, win/loss detection, and pause/resume lifecycle. A React `TwentyFortyEightView` component renders the board with a neon palette and handles keyboard input and universal overlays.

## Tech Stack

TypeScript, React, CSS Grid, existing Kali Toys core (`BaseGame`, `GameRegistry`, `GameWindow`), existing `GameStatus` / `ActionType` / `GameCommand` constants.

## Global Constraints

- Extend `BaseGame` from `src/games/core/base-game.ts`.
- Register via `GameRegistry.register(GameType.TWENTY_FORTY_EIGHT, TwentyFortyEightGame)`.
- Use existing constants; never redefine strings.
- State lives in the game class; the view reads `game.getState()` and triggers React re-renders only when status/score change.
- Build passes `npx tsc --noEmit` with zero errors.

### Task 1: TwentyFortyEightGame class

**Files:**
- Create: `kali-web/src/games/twenty-forty-eight/twenty-forty-eight-game.ts`

**Behavior:**
- Board sizes: 3×3, 4×4, 5×5, 6×6 (configurable via `rules.size`).
- Start with two random tiles (90% `2`, 10% `4`).
- Handle `UP` / `DOWN` / `LEFT` / `RIGHT` moves.
- Merge adjacent equal tiles; each merge adds the merged value to the score.
- Spawn one new tile after each successful move.
- Detect win when any tile reaches `2048`.
- Detect loss when no empty cells and no merges are possible.
- Support universal commands: `start`, `restart`, `pause`, `resume`, `give_up`, `play_again`.

**State data shape (`BoardData`):**
```typescript
interface BoardData {
  size: BoardSize;
  cells: (Tile | null)[][];
  score: number;
  bestTile: number;
  moves: number;
  won: boolean;
  over: boolean;
  lastSpawned: { row: number; col: number } | null;
  mergedIds: number[];
}
```
